# guide-mzh-aiyouxi

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容互不关联，各自独立展示，不针对任何特定域名或网站。

## 项目简介

这是一个以归档形式存放静态 HTML 页面的仓库。仓库本身不提供任何持续更新或维护服务，仅作为历史页面的存储和展示空间。

## 目录说明

```
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的公共资源（如样式、脚本、图片等）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立页面。
- `assets/`：存放页面可能共用的静态资源，便于统一管理。

## 页面归档说明

所有页面均为历史存档，内容可能不再更新或维护。页面中可能存在过时信息、失效链接或不再使用的功能。使用者请自行判断页面的适用性。

## 维护说明

本仓库不定期更新，不保证所有页面持续可用或内容及时更新。如发现页面显示异常，可提交 Issue 反馈，但不保证修复。

## 使用方式

可直接通过 GitHub Pages 或其他静态托管服务访问 `pages/` 目录下的 HTML 文件。具体访问方式请参考托管平台文档。

## 许可证

本仓库内容仅供学习与参考，不构成任何形式的使用授权或责任担保。具体页面的版权归属请参考页面内标注。
